about Java script

Java script is like the magician of the web world!

It is a scripting language that adds interactivity and dynamic features to website, making them more engaging for users.

uses of java script

Java script can be used for various purposes such as creating interactive forms,dynamic content updates,animations,games and even building entire web applications.
It allows you to respond to user actions,manipulate the content of a webpage and communicate with servers to fetch
or send data without needing to reload the page
